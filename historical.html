Timeline Sejarah Javascript

Tahun Perkembangan
1995 JavaScript dibuat oleh Brendan Eich di Netscape sebagai bahasa skrip ringan untuk halaman web.
1997 ECMAScript, standar dasar JavaScript, pertama kali diterbitkan. JavaScript 1.1 juga diperkenalkan.
2002 Pengembangan ECMAScript 4 dimulai dengan tujuan peningkatan signifikan, tetapi proyek ini akhirnya ditinggalkan.
2005 AJAX (Asynchronous JavaScript and XML) menjadi populer, memungkinkan pembaruan halaman web secara dinamis dan asynchronous.
2009 ECMAScript 5 (ES5) dirilis, memperkenalkan fitur baru dan perbaikan pada bahasa.
2015 ECMAScript 2015 (ES6) membawa perubahan besar, termasuk fungsi panah, kelas, dan modul, menandai pergeseran ke JavaScript yang lebih modern.
2016-2019 Pembaruan tahunan ECMAScript (ES7 hingga ES10) berlanjut, menambahkan fitur seperti async/await, operator spread/rest, dan chaining opsional.
2017 Rilis Node.js 8.0 membawa dukungan yang luas untuk fitur ES6 ke sisi server JavaScript.
2018 ECMAScript 2018 (ES9) memperkenalkan fitur seperti iterasi asynchronous dan properti rest/spread.
2019 ECMAScript 2019 (ES10) menambahkan Array.prototype.flat(), Array.prototype.flatMap(), dan perbaikan lainnya.
2020-2022 Pembaruan tahunan ECMAScript berlanjut membawa fitur dan perbaikan baru, seperti nullish coalescing (??) dan optional chaining (?.).
2023 Pertumbuhan terus-menerus dari framework dan perpustakaan JavaScript, dengan React, Angular, dan Vue.js tetap populer untuk pengembangan front-end.
